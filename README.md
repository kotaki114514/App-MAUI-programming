# StudyApp (MAUI Project)

##  Overview
This is a .NET MAUI application designed for student learning management.

##  Features
- Home dashboard
- Notes system (CRUD)
- Profile page
- Check-in system
- Local database storage

##  Tech Stack
- .NET MAUI
- C#
- MVVM pattern
- SQLite

##  How to Run
1. Open StudyApp.sln in Visual Studio 2022
2. Restore NuGet packages
3. Run on emulator or Windows
